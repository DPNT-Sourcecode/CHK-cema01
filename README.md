# tdl-warmup-python


## 1. Requirements

- `Python 3.12`
- `pip` (ensure it supports `Python 3.12`)


## 2. How to start

- Install dependencies `pip install -r requirements.txt`
- Open `lib/send_command_to_server.py`
- Read the comments as documentation, they will guide through the rest of the setup
